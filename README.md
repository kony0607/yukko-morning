## 🐛 エラーが出た場合

### `EADDRINUSE: address already in use :::3000`

別のプロセスがポート 3000 を使用中。解決：

```bash
# ポート確認
lsof -i :3000

# 強制終了（必要に応じて）
kill -9 <PID>

# 別ポートで起動（server.js 編集）
# const PORT = 3001;
```

### `Cannot find module 'express'`

依存パッケージが不足。解決：

```bash
npm install
npm start
```

### `SQLITE_CANTOPEN`

DB ファイルが読み込めない。解決：

```bash
# DB 削除
rm orders.db

# 再起動で自動作成
npm start
```

## ⚠️ 注意事項

### **ローカルネットワーク内のみ対応**

現在は LAN（同じルータに接続）のみ対応。
インターネット公開する場合は HTTPS 化とセキュリティ設定が必要。

### **データはブラウザメモリにもキャッシュされる**

- ページリロード → サーバーから再読み込み
- インターネット遮断時 → ローカルキャッシュで続行（部分的）

### **複数タブを同時リロードすると同期ズレが起きる可能性**

リロード後に手動で「集計」タブを開いて確認推奨。
